**TTX-Monitor+** es un cargador de software para la máquina arcade **Type X** de **Taito**.
Desarrollado originalmente por **Romhack**: **[xb_monitor](https://github.com/zxmarcos/ttx_monitor)**

<br />

Algunas funcionalidades fueron añadidas/cambiadas/eliminadas:

<br />

- Añadido soporte para **AxisR**, **AxisZ**
- Añadida la opción **PovAsAxis**
- Corregida la sensibilidad de la zona muerta
- Añadida la función estándar **SavePatch**
- Nueva interfaz de ususario para la configuración
- Eliminado el wrapper de **Direct3D**
- Desactivado el motor de registro

<br />

La característica nueva más destacable es la implementaciñon de un wrapper de **DirectInput** con soporte para los títulos de **mahjong**.
