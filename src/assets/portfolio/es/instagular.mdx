**Instagular** es un cliente web personalizado para Instagram, con el objetivo de ofrecer una mejor experiencia de escritorio, en contraste al diseño para dispositivos móviles que usa el sitio oficial, además de contar con funcionalidades adicionales no presentes en la experiencia regular de Instagram.

<br />

Este proyecto implementa su propio servidor backend API utilizando un fork de [instagram-private-api](https://github.com/Hipnosis183/instagram-private-api) para conectarse con Instagram, actuando como intermediario entre el cliente de Instagular y el servidor de Instagram.

<br />

## Funciones implementadas

- Administración de sesiones.
- Feeds con manejo de paginación automática.
- Visor multimedia a pantalla completa, reproduciendo contenido con mayor calidad de imagen.
- Perfiles de usuario (posteos, reels, videos, etiquetados).
- Historias de usuario y destacadas.
- Interacción entre usuarios (likes, seguir, comentarios).
- Descargar contenido multimedia en calidad alta/original.
- Elementos guardados y colecciones.
- Motor de búsqueda de usuarios.
