**Glaucous** es un **constructor de base de datos de videojuegos**, **administrador de librería** y **frontend para emuladores**. Te permite crear tu base de datos personalizada, además de la posibilidad de utilizarlo como un lanzador/frontend.

<br />

Tanto la aplicación como la base de datos están diseñadas para ser lo más dinámicas posible, permitiendo elegir que querés ver y cómo. **Personalización** y **flexibilidad** son objetivos clave, permitiéndote adaptar tu configuración actual sin forzar cambios específicos para funcionar.
